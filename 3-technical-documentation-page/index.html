What is Static Analysis

It is tooling that scrutinizes code without running it, as opposed to "dynamic analysis" which tests the result of running code - like testing (unit tests or large end-to-end tests).

What is Formatting

A formatter is a tool that

  1. reads your source code
  2. ignores your formatting
  3. rewrites it based on config rules

Ex. Prettier

What is a Linter

A linter is a tool that runs a set of checks on your source code

  1. Each check is a rule
  2. Each rule may report on code it doesn't like
  3. Each complaint may contain an optional autofix
